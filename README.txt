# Клінічна оцінка лікарського засобу Гастритол (Gastritol_Clinical_Trial)

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.XXXXXXX.svg)](https://doi.org/10.5281/zenodo.XXXXXXX) [![R](https://img.shields.io/badge/R-%E2%89%A5%204.0.0-blue.svg)](https://www.r-project.org/)
[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)

Цей репозиторій містить повний набір R-скриптів для відтворення статистичного аналізу та візуалізації даних клінічного дослідження щодо оцінки ефективності та безпеки лікарського засобу "Гастритол" при лікуванні функціональної диспепсії. 

Дані та програмний код публікуються у відкритому доступі з дозволу компанії-спонсора для забезпечення максимальної прозорості, відтворюваності результатів та дотримання стандартів відкритої науки (Open Science).

## 📂 Структура репозиторію

Для коректної роботи скриптів репозиторій має наступну структуру. Папки `results/tables` та `results/figures` створюються автоматично під час виконання коду.

```text
Gastritol_Clinical_Trial/
├── data/
│   ├── raw/                           # Вихідні знеособлені дані у форматі Excel
│   └── processed/                     # Очищені та стандартизовані дані (.csv)
├── results/
│   ├── tables/                        # Згенеровані таблиці (Table 1-4) (.xlsx)
│   └── figures/                       # Згенеровані графіки (Figure 1-4) високої роздільної здатності (.png)
├── README.md                          # Опис проєкту
└── scripts/                           # Набір R-скриптів (див. опис нижче)
🛠 Опис аналітичного пайплайну (Скрипти)
Аналіз розділений на 6 послідовних модулів, кожен з яких виконує специфічну задачу:

01_data_preparation.R

Зчитування сирих даних (readxl).

Очищення, стандартизація назв змінних (перехід на латиницю), парсинг шкал Лікерта.

Факторизація категоріальних змінних та збереження підготовленого датасету clean_gastritol_data.csv.

02_baseline_custom_stats.R

Аналіз базових демографічних та клінічних характеристик пацієнтів (Day 1).

Автоматизована перевірка на нормальність розподілу (критерій Шапіро-Уїлка).

Застосування t-критерію Стьюдента або U-критерію Манна-Уїтні залежно від розподілу.

Розрахунок точних 95% довірчих інтервалів (метод Клоппер-Пірсона) для біноміальних розподілів (стать).

Вихідний файл: Table_1_Baseline_Stats.xlsx.

03_efficacy_analysis.R

Оцінка клінічної ефективності через розрахунок динаміки симптомів (Day 1 → Day 21) та обчислення показника "Дельти" (Δ).

Внутрішньогруповий аналіз (критерій Вілкоксона для пов'язаних вибірок).

Міжгруповий аналіз ефективності за Дельтою (критерій Манна-Уїтні).

Вихідний файл: Table_2_Efficacy_Analysis.xlsx.

04_patchwork_style_viz.R

Побудова комплексних багатопанельних візуалізацій для 6 ключових клінічних симптомів.

Використання комбінації geom_boxplot та geom_jitter для прозорої репрезентації розподілу індивідуальних значень.

Об'єднання панелей за допомогою пакета patchwork.

Вихідний файл: Figure_1_Custom_Style.png.

05_safety_and_risk_analysis.R

Аналіз профілю безпеки та прихильності до лікування (комплаєнс).

Епідеміологічний аналіз клінічного успіху: розрахунок Різниці ризиків (RD), Відносного ризику (RR), Відношення шансів (OR) та Number Needed to Treat (NNT) із 95% ДІ.

Генерація Forest Plot для представлення логарифмічної шкали відносних ризиків.

Вихідні файли: Table_3_Safety_and_Compliance.xlsx, Table_4_Risk_Metrics.xlsx, Figure_2_Clinical_Success.png, Figure_3_Forest_Plot.png, Figure_4_Safety_Compliance.png.

06_consort_flow_diagram.R

Програмна генерація стандартизованої блок-схеми дизайну дослідження за правилами CONSORT.

Побудова повністю засобами ggplot2 без використання зовнішніх графічних редакторів.

Вихідний файл: Figure_1_CONSORT_Flow_Diagram.png.
